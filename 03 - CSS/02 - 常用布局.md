## 水平垂直居中

##### 行内元素（文字、图片等）

```css
.father{
  text-align: center;
}
.son {
  height: 20px;
  line-height: 20px;
}
```

##### 块级元素

1：绝对定位 + margin（需要指定子元素的宽高，不推荐）

```css
.father{
  position: relative;
  min-height: 500px;
  background: pink;
}
.son {
  position: absolute;
  width: 200px;
  height: 100px;
  background: red;
  top: 50%;
  left: 50%;
  margin-top: -50px;
  margin-left: -100px;
}
```

2：绝对定位 + translate（无需指定子元素的宽高，推荐）

```css
.father{
  position: relative;
  min-height: 500px;
  background: pink;
}
.son {
  position: absolute;
  background: red;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```

3：flex 布局（所有子元素均局中）

```css
.father{
  display: flex;
  justify-content: center;
  align-items: center;
}
```

4：flex 布局 + margin: auto（推荐）

```css
.father{
  display: flex;
}
.son {
  margin: auto;
}
```
