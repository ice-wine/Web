#### VS Code快捷键

mac为例：

1.移动光标 

​	cmd(单行)/option(单词) + 方向键

2.删除

​	cmd(单行)/option(单词)  + delete：光标前后

​	cmd + shift + K：整行

3.编辑

​	cmd + enter：当前光标下增加一行

4.JS

​	cmd + /：注释

​	option + shift + F：格式化

​	ctrl + J：将多行代码合并为一行

5.跳转

​	ctrl + tab：切换标签
